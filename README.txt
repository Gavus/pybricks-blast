Import all the files into "Pybricks Code" IDE.

blast.py: Defines the Blast robot.
demo.py: Run Blast robot and showing off what it can do.
keyboard.py: Control Blast robot using the keyboard instead. See the file for which keys are being used.

challenge.py: The team should edit this file.

main.py: The main function is defined here. Change here if you want to use run_demo, run_keyboard, or run_challenge.
